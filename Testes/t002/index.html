Conceitos Fundamentais

Estrutura Mínima de um Documento HTML

Todo HTML começa com <!DOCTYPE html> seguido das tags <html>, <head> e <body>.

Tags de Texto

Use <h1> a <h6> para títulos, <p> para parágrafos, <br> para quebra de linha e <strong> para destacar.

Listas

Você pode usar listas ordenadas <ol> ou não ordenadas <ul> com itens <li>.

Mídia

Imagens usam a tag <img src="" alt="">, vídeos <video> e áudio <audio>.

Semântica

Use tags como <header>, <main>, <article>, <section> e <footer> para dar significado ao conteúdo.
